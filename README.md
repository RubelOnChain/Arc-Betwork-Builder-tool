# Arc Network Privacy & Analytics Dashboard 🌌

An interactive, tech-noir inspired analytics and utility dashboard built for the **Arc Network** community. This application provides users with conceptual tools to validate transaction privacy and check for network sybil activity.

## 🌐 Live Application
Check out the fully functional deployment here: 
👉 **[https://arc-synth-scan.lovable.app](https://arc-synth-scan.lovable.app)**

---

## 🛠️ Key Features
- **Zk-Privacy Validator:** Conceptual interface to verify secure state transactions on Arc infrastructure.
- **Sybil Guard Simulator:** A tool to flag multi-account or engagement-farming wallet addresses to keep the ecosystem organic.
- **Live Network Metrics:** High-fidelity monitoring of active node distribution and cryptographic states.

## 🚀 Tech Stack & Setup
- **Frontend:** React, Vite, Tailwind CSS
- **Theme:** Cyberpunk / Tech-Noir (Neon Violet & Cyan)
